# Contributing to weekly-builds-community

Thanks for being here. There are two ways to contribute:

---

## 1. 🎮 Play the Weekly Progress Game

The easiest way to get started. Every week, share what you built.

1. Fork this repo
2. Open `progress/YYYY-WWW.md` for the current week (create if missing)
3. Add your entry using the template in [`WEEKLY_PROGRESS.md`](./WEEKLY_PROGRESS.md)
4. Open a PR titled: `progress(W17): your name — one line summary`

Top contributors get tagged every Monday in the Weekly Issue and on LinkedIn.

---

## 2. 🛠️ Improve the Repo Itself

1. Check [open issues](https://github.com/P-r-e-m-i-u-m/weekly-builds-community/issues)
2. Comment to claim before starting
3. Fork → branch → PR

**Branch naming:**
```
feat/your-feature
fix/what-you-fixed
docs/what-you-updated
```

---

## PR Checklist

- [ ] Followed the entry template
- [ ] One entry per person per week
- [ ] PR title follows the format
- [ ] No unrelated files changed
