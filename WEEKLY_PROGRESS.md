# 🎮 Weekly Builds Community — The Game

> Build in public. Get recognized. Grow together.

This is an open contributor game. Every week, developers from anywhere drop their progress here via a PR — what they built, what blocked them, and what's next. Every Monday, top contributors get a **public shoutout** in the Weekly Issue and on LinkedIn.

The more you show up, the more visible you get.

---

## 📋 How to Play

1. **Fork** this repo
2. Open `progress/YYYY-WWW.md` for the current week (create it if it doesn't exist)
3. Add your entry using the template below
4. Open a PR titled: `progress(W17): your name — one line summary`

No setup. No new tools. Just GitHub.

---

## 📝 Entry Template

```markdown
### [Your Name](https://github.com/your-handle) — Week WXX

**What I built / worked on:**
<!-- 1-3 sentences. Be specific. -->

**What blocked me:**
<!-- Honest blockers. "Nothing" is fine if true. -->

**What's next:**
<!-- What you're tackling next week -->

**Links:** <!-- PR, repo, deploy — optional but encouraged -->
```

---

## 🏆 Recognition Rules

- Top **3 contributors** each week get tagged in the Monday issue + LinkedIn post
- Consistency matters — showing up 4 weeks in a row earns a **🔥 Streak** badge
- Quality over quantity — one great entry beats five empty ones

---

## 📁 File Structure

```
progress/
  2026-W17.md
  2026-W18.md
```

Each file covers one ISO week. Multiple contributors add entries via PR.

---

## ⚠️ Rules

- One entry per person per week
- Must follow the template format
- No spam or self-promotion without substance
- PRs that skip the template will be closed with a friendly note

---

## 📅 Weekly Cycle

| Day | What happens |
|-----|-------------|
| Mon | Weekly Issue opens, top contributors tagged |
| Mon–Sun | PRs accepted for that week |
| Sun midnight | Entries close for the week |
